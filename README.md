# Sandbase Homebrew tap

Homebrew tap for the [Sandbase](https://sandbase.dev) CLI (`sand`).

```bash
brew install sandbasehq/tap/sand
```

Or:

```bash
brew tap sandbasehq/tap
brew install sand
```

Then:

```bash
sand login
sand --help
```

Other install methods: `curl -fsSL https://sandbase.dev/install.sh | bash` or `npm install -g @sandbase/cli`.
